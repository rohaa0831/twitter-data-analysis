# twitter-data-analysis
## Objectives
-> Learn how to preprocess large datasets into useful DataFrames, from which interesting insights could be made about Donald Trump's twitter activity.

-> Work with data visualization libraries matplotlib.pyplot and seaborn to create figures that illustrate the variance in Trump's twitter activity depending on the time of day during the years 2016-2018.

-> Constructed a PCA (Principal Compositoin Analysis) heatmap to display some of Trump's most polarizing words choices utilizing Vader Lexicon's word polarity matrix.
