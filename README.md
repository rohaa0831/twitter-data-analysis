# twitter-data-analysis
## Objectives
-> Learn how to preprocess large datasets into useful DataFrames, from which interesting insights could be made about Donald Trump's twitter activity.

-> Work with data visualization libraries matplotlib.pyplot and seaborn to create figures that depict Trump's twitter behavior from 2016-2018. 

-> Constructed a PCA (Principal Compositoin Analysis) heatmap to display some of Trump's most polarizing words choices utilizing vade lexicon's word polarity matrix.
